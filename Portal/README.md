CV-F2025 Homework Portal
========================

How to run
----------
1) Create and activate a virtual environment (recommended).
2) Install requirements:

   pip install -r requirements.txt

3) Start the app:

   python app.py

4) Open http://127.0.0.1:5000 in your browser.

Notes
-----
- HW2 and HW4 require OpenCV and can be compute-heavy on large images.
- HW3 Part 5 will attempt to use ultralytics SAM2 if installed. If not, upload precomputed masks.
- Static demos for HW1 and HW7 do not need any backend.
